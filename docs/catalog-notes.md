# Catalog Notes

`catalog/graffiti.json` is the list the plugin loads for the menu and random picker.

Keep the file in the plugin folder:

```text
csgo/addons/counterstrikesharp/plugins/RasElHanoutGraffiti/catalog/graffiti.json
```

After editing it on a running server:

```text
css_grafreload
```

The plugin ignores duplicate definition indexes and keeps the first valid one it finds. Keep names short enough for the in-game menu.
