# RasElHanout Graffiti

CS2 graffiti selector for CounterStrikeSharp.

This is the one used on Ras El Hanout DZ: three slots, weekly uses, VIP/admin limits, menu picker, website picker link, and vanilla spray replacement.

## Install

Copy `addons` into your server `csgo` folder.

Load it:

```text
css_plugins load RasElHanoutGraffiti
```

The config is generated on first load. Set the database info there, then reload the plugin.

Source is in `src`.
